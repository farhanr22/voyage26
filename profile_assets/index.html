Voyage 2k25 Attendee Profiles

This site hosts individual static profile pages for attendees of the Voyage 2k25 departmental fest and is served via Netlify from the repository’s deploy branch.
Each page (e.g., /ABCD.html) is generated by a GitHub Action periodically triggered by the backend when any pages need to be updated. The Action requests the latest data from the backend API endpoint, which returns a JSON response after verifying the attached API password.
